## **Wbooru**

### **简介**
此软件是用于方便浏览某些图片画廊。**此项目还在开发中，但基本功能基本实现，请ReleaseOnly/Star关注:D**

### 支持的功能
* 功能插件化
* 支持标签和图片搜索
* 本地浏览/收藏/下载图片
* 快速打开图片画廊网站，快速操作图片
* 收藏/过滤标签，支持订阅标签的图片
* 支持界面全程单手操作
<br>etc...
